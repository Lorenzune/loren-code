# Code Of Conduct

## Our Standard

We want Loren Code to be a friendly, practical project where people can ask questions, report problems, and contribute without being treated badly.

Please be respectful, constructive, and patient.

## Expected Behavior

- be kind and professional
- assume good intent
- give clear, honest feedback without being hostile
- focus on improving the project, not winning arguments
- help keep discussions welcoming to beginners and experienced contributors alike

## Unacceptable Behavior

- harassment, abuse, or personal attacks
- hateful or discriminatory language
- intimidation or repeated bad-faith behavior
- sharing private information without permission
- intentionally disruptive trolling or spam

## Scope

This applies to project discussions, issues, pull requests, and other public community spaces related to Loren Code.

## Enforcement

If something goes wrong, open a private report through GitHub security/contact channels or reach out through the repository owner account.

Project maintainers may remove comments, close threads, or block contributors when necessary to protect the community.
